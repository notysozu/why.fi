# Plan 02-02: Build heuristic functions for "Why Meter" and Emoji matching

<objective>
Develop mathematical heuristics based on the 468 facial landmarks to evaluate specific emotions and the aggregate "Why Meter" score.
</objective>

<action_plan>
<task>
<read_first>
- backend/ml_engine.py
</read_first>
<action>
In `backend/ml_engine.py`, add Euclidean distance helper functions.
Add methods to calculate:
- Mouth Aspect Ratio (MAR) (for open/closed mouth)
- Eye Aspect Ratio (EAR) (for wide/closed eyes)
- Eyebrow position relative to eyes.
Create a method `classify_emoji(landmarks)` that uses these metrics to return one of: "happy", "sad", "angry", "surprise", or "neutral".
Create a method `calculate_why_meter(landmarks)` that returns a dictionary with scores (0-100) for "boredom", "confusion", "dread", and an overall "why_score".
Update `process_frame` to return `{"emoji": emoji, "why_meter": why_scores}` instead of just raw landmarks.
</action>
