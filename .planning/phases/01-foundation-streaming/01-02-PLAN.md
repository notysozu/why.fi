# Plan 01-02: Implement WebRTC capture on the frontend

<objective>
Capture live video from the user's webcam via WebRTC and display it on screen.
</objective>

<action_plan>
<task>
<read_first>
- frontend/src/App.tsx
</read_first>
<action>
Create `frontend/src/components/WebcamCapture.tsx`.
Use `useRef` for a `<video>` element.
In a `useEffect`, call `navigator.mediaDevices.getUserMedia({ video: true })` and assign the stream to the video element's `srcObject`.
Handle errors if the user denies permission (display a simple error text).
Update `App.tsx` to render `<WebcamCapture />` and remove default Vite boilerplate.
</action>
<acceptance_criteria>
- `frontend/src/components/WebcamCapture.tsx` contains `navigator.mediaDevices.getUserMedia`.
- `frontend/src/App.tsx` imports and uses `WebcamCapture`.
</acceptance_criteria>
</task>
</action_plan>
